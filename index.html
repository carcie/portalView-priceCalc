Formalización del Modelo Matemático

Definiciones técnicas y base algebraica del sistema de precios.

v1.1

Modelo A: Estándar de Ventas

Principio: Cada proyecto se cotiza con capacidad independiente para proteger márgenes y evitar desbordamiento.

Fórmula de Precio Piso (Pfloor,i)
Pfloor,i = Vi + Oi + Li + Falloc,i

Donde:
• Vi = 217 · λi · di (Visitas)
• Oi = 0.50 · oi (Objetos)
• Li = 6,200 · ⌈ ai / 10,000 ⌉ (Licencia)
• Falloc,i = (di / 12) · (Fbase / Ntarget)
Nota Estratégica: El uso de la función techo (⌈...⌉) a nivel de proyecto individual asegura que el cliente pague por el bloque completo de 10k m² que activa.

Modelo B: Estándar Operativo

Principio: Los costos reales se basan en la demanda agregada del portafolio, aprovechando economías de escala.

Fórmula de Costo Real (Cactual)
Cactual = Fbase + Lagg + ∑(Vi + Oi)

Donde:
• Fbase = 43,360 + A (Fijos)
• Lagg = 6,200 · ⌈ (∑ai) / 10,000 ⌉
• ∑Vi = Suma de costos variables de visitas
Ganancia por Arbitraje: La diferencia entre cobrar Li (segregado) y pagar Lagg (agregado) genera un excedente operativo.
